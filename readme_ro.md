## Description

Nest este un cadru de dezvoltare ce permite crearea de aplicații server de înaltă performanță, cu un flux de lucru rapid și eficient, capabile să ruleze pe <a href="https://nodejs.org" target="_blank">Node.js</a>, mediu de execuție al limbajului de programare JavaScript. Acesta ține cont de cele mai moderne tendințe din domeniul data utilizând <a href="https://www.typescriptlang.org" target="_blank">TypeScript</a> un superset sintactic al limbajului de programare JavaScript (păstrează compatibilitatea cu JavaScript pur) și combină elemente de OOP (Programare orientată pe obiecte), FP (programare funcțională) și FRP (programare reactivă funcțională).

<p>Sub capotă, Nest folosește <a href="https://expressjs.com/" target="_blank">Express</a>, dar în același timp este compatibil cu o gamă largă de pachete, precum <a href="https://github.com/fastify/fastify" target="_blank">Fastify</a>, fapt ce permite utilizarea ușoară a multitudinii de pluginuri terțe disponibile.</p>

## Filozofie

<p>În ultimii ani, datorită mediu de execuție Node.js, JavaScript a devenit „Lingua Franca” a Web ului atât pentru aplicații frontend, cât și pentru cele backend, dând naștere unor proiecte minunate precum <a href="https://angular.io/" target="_blank">Angular</a>, <a href="https://github.com/facebook/react" target="_blank">React</a>, și <a href="https://github.com/vuejs/vue" target="_blank">Vue</a>, care duc la creșterea productivității dezvoltatorilor și permit construcția de aplicații frontend rapide, testabile și extensibile. Cu toate acestea, pe partea de server, în timp ce există o mulțime de pachete, cadre de dezvoltare superbe și instrumente pentru Node.js, niciunul dintre ele nu rezolvă eficient problema principală - arhitectura.</p>
<p>Nest își propune să ofere o arhitectură de aplicații din cutie, care să permită crearea fără efort a aplicațiilor extrem de testabile, scalabile și ușor de întreținut. Drept sursă de inspiratie, în termeni de arhitectură, a sevit Angular.</p>
